# 5. Installation

In general clone repository:

```bash
git clone https://github.com/Jamro90/PICOntroller.git
```

Next navigate:

```bash
cd PICOntrller
```

and build project:

```bash
make install
```

You can also skip building and run it from source:

```bash
make
```

or 

```bash
python3 main.py
```
